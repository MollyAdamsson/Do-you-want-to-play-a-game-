Rock, Paper, Scissors, Lizard, Spock is a game of chance that expands the traditional game of Rock, Paper, Scissors.

The main idea of this game is the same as with an ordinary round of rock, paper, scissors. The only difference is that this game has some extra options for you to choose between.

Click on the button that says "Start Game" to set off the timer.
You then have 30 seconds and 5 rounds to finish the game.
